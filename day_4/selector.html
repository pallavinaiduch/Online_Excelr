SELECTOR

universal selector

it indicates with *

it select all elements

grouping selector

it is used to selected group of elements
it is indicated with comma(,)

combined selector

descending selector

it selects all child elemnts (direct/indirect)

            parent child{
                 // both direct and indirect child will select
            }

child selector

it selects direct child elements

            parent > child{
                // it will direct child
            }

adjacent sibling selector

it select immidiate adjacent element

            main + adajacentelement{
                // it selects immidiate adjacent element
            }

general sibling selector

it selects all adjacent elements

            main ~ adjacentelement{
                // it selects all siblings to the main elements
            }
